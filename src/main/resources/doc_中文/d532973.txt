耀晶片 更让怪物们情何以堪的还有空之轨迹里的耀晶片奖励,刷过羊的玩家们你们懂的!想当年玩第一作的时候,小编我总是用N场战斗辛辛苦苦把小艾的珠子点燃,憋死不用,专等这个奖励出现,然后立马按下珠子插楼,一阵金箍棒乱舞(观众:你以为是孙悟空啊?!)。于是只见各种颜色的耀晶片哗啦啦的掉,清脆的声音绵延不绝,听起来真是销魂啊…… 怪物和怪物不得不说的故事 而勇者斗恶龙更把“合成”提升到一个新的台阶:怪物合成!于是怪物也可以拿来当“材料”,合成更强力的怪物。小编不得不感叹:地球已经阻止不了“合成”了!不知道一个战士、一个法师、一个奶妈等等等等,能不能合成一个超级无敌赛亚人? 软星 日式RPG大搞怪物合成,国产RPG当然也不甘示弱,仙剑奇侠传系列的炼蛊皿就是一个不错的东东 (微博) ,炼出毒虫砸BOSS,各种中毒各种晕,哈哈!不过呢,到了仙剑四,炼蛊被锻造和注灵取代,而注灵材料“软星”,请问小编可以理解为这是研发团队在自我吐槽么…… 胖子 作为国产RTS游戏神作,98年的《铁甲风暴》(这不是《血狮》!)凭借它完美的DIY合成系统吸引了无数硬派游戏爱好者。到现在为止蜂刺II的形象小编还记忆犹新,至于这个胖子……你能看出它哪里胖了吗? 史莱姆凝胶 说到合成就不得不提terraria,作为minecraft的升级版,它完美的继承了minecraft的所有玩点,依然无边的世界、依然轻松的氛围、依然100%的自由。加上更加强大的合成系统以及更加怀旧的2D横版风格画面,terraria显得更加亲和,不过,不得不说的就是,越庞大的合成表意味着越多的奇怪材料。没错,“史莱姆凝胶”、“眼球胶体”、“陨石碎片”在向你招手有木有! 好人卡 如果说单机游戏里玩合成是自娱自乐,那么网游里玩合成就是为了强大、强大、更强大!当然,也是为了和兄弟们朋友们炫耀,嘿嘿!看我头上这稀有称号,多帅气啊!你问我哪来的?符文卡换的。符文卡哪来的?用9张好人卡合成的……囧啊,梦幻诛仙(微博) 你赢了…… 妈妈的诚心,爸爸的青春…… 前边列的那些材料名称,还都只是一个两个搞笑而已,简直是弱爆了,像希望OL这样吐槽点满满的材料列表才够给力!看,“妈妈的心”还有升级版“妈妈的诚心”,这个系列绝对可以打造成名牌;至于“爸爸的青春”“爸爸的解释”,凭名称已经完全搞不懂是啥玩意儿了……还有那个“男人的痴呆症”是肿么回事啊,谁来给个解释?爸爸你来解释下! 腾讯网首页 | 网站地图 QQ邮箱: 未读邮件 ( ) 漂流瓶 ( ) 群邮件 ( ) 文件夹 ( ) QQ空间: 我的动态 ( ) 好友动态 ( ) 我的参与 ( ) 您好, [ 退出 ] | 新闻 视频 图片 评论 财经 股票 港股 基金 娱乐 明星 电影 音乐 体育 NBA 中超 彩票 汽车 车型 房产 家居 科技 数码 手机 家电 时尚 健康 美容 育儿 服饰 购物 旅游 杂志 文化 教育 小说 出国 游戏 动漫 漫画 星座 大家 微博 军事 论坛 公益 医疗 儿童 天气 游戏 > 游戏新闻 > 滚动新闻 > 正文 2011年11月01日13:28 腾讯游戏 [ 微博 ] 我要评论 ( 0 ) 字号: T | T 转播此图到微博 新功能 幻灯观看 重新播放 转播到微博 你可能还喜欢 隐藏 查看图注 进入游戏频道首页 查看原图 已有 183536852 条与 游戏 分享到: 腾讯微博 QQ空间 微信 朋友 新浪微博 人人 QQ邮箱 开心网 我要评论( 0 ) 相关阅读: · 2011.11.01 · 2011.11.01 · 2011.11.01 · 2011.11.01 · 2011.11.01 推荐微博: 腾讯游戏频道 腾讯网;游戏 腾讯爱玩 腾讯爱玩游戏平台 剑灵 知名RPG网游 魔兽世界 魔兽世界官方微博 魂之猎手 MF官博 全部收听 注册微博 [责任编辑:ghosthuang] 网友评论: 已有 0 条评论,共0人参与,点击查看 登录 (请登录发言,并遵守 相关规定 ) 分享至:
